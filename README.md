# Tessa's Dev Blog

A personal developer blog where I share what I've learned — built with the [Fuwari](https://github.com/saicaca/fuwari) Astro template and deployed on GitHub Pages.

## About

This site is my space to document and share things I discover as a developer: technical deep-dives, lessons learned, and notes on tools and concepts I find useful. All posts are written by me.

## Tech Stack

- **Framework**: [Astro](https://astro.build/)
- **Template**: [Fuwari](https://github.com/saicaca/fuwari) by [@saicaca](https://github.com/saicaca)
- **Styling**: Tailwind CSS + Svelte
- **Search**: Pagefind
- **Deployment**: GitHub Pages

## Getting Started

### Prerequisites

- Node.js
- pnpm

### Install & Run

```bash
pnpm install
pnpm dev
```

### Build

```bash
pnpm build
```

### Create a New Post

```bash
pnpm new-post
```

## Customizations from Original Template

- Profile and navigation bar config
- Custom banner image
- Theme color adjustments
- Posts written from scratch
